### `lester-tools js工具库说明`

📦 安装（Install）
#### npm install lester-tools
or
#### yarn add lester-tools

🔨 Usage  
import { setTitle, copy } from 'lester-tools';
>  setTitle('页面标题')  
>  copy('复制文本')
